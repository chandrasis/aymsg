Class AYmsgBodyBuffer

  • java.lang.Object
    • org.if_itb.aymsg.AYmsgBodyBuffer
public class AYmsgBodyBuffer
extends java.lang.Object
This handy class hides most of the pain of building the Yahoo message body. Each body consists of key/value pairs (or sometimes just keys) with each field separated by the sequence 0xc080 Note: this class is NOT thread safe (although, to be honest, building a single message body from more than one independent thread is surely asking for trouble?!??)
